# Synthesis and Voice Conversion Reference Scripts

These scripts are kept for later V2/V3 experiments:

- `synthesize_zhuang_tts.py`: generate `facebook/mms-tts-zyb` audio from text.
- `convert_openvoice_v2.py`: convert synthesized speech to multiple reference voices.
- `audio_compare_app.py`: local browser app for listening comparisons.
- `build_audio_demo.py`: static audio demo builder.

They are not part of V1 clean data generation.
